## Telegram Mini App Setup

1. Створіть бота через @BotFather (`/newbot`) або використайте існуючого.
2. Налаштуйте кнопку запуску Mini App:
   - Відкрийте @BotFather
   - Виконайте `/mybots` -> виберіть бота -> `Bot Settings` -> `Menu Button` -> `Configure menu button`
   - URL: `https://electricalengineergeek-svg.github.io/AiT/app/`
3. За потреби додайте коротку команду запуску (`/start`) і текст з посиланням на Mini App.
4. Відкрийте вашого бота в Telegram і натисніть кнопку меню, щоб запустити застосунок.

### Рекомендації

- Після змін у мета-тегах або прев'ю сторінки враховуйте кеш Telegram.
- Якщо потрібно оновити прев'ю швидше, тимчасово використайте URL із параметром версії, наприклад `?v=2`.
